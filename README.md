# TrigonometricFunctions
2021年春季现代软件工程小组作业仓库

## 实现
三角函数功能计算器,对三种基础的三角函数及反函数实现计算功能



## 运行指南
### 依赖
- python 3.6+
- Qt PyQt5.QtCore PyQt5.QtWidgets
- UI设计采用docker：https://github.com/jozo/docker-pyqt5

### 运行
```shell
$ python TrigonometricFunctions\TriCalculatorUi\CalculatorUi.py
```
